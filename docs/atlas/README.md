## System

```mermaid
flowchart LR
  Customer((Customer))
  Agent((Agent))

  Customer -->|portal form| Portal[Portal · Next.js]
  Customer -.->|sends email| Gmail[(Support inbox<br/>e.g. Gmail)]
  Agent --> Bridge[Bridge · Next.js]

  Portal --> API[API · NestJS]
  Bridge --> API

  Gmail -.->|Gmail REST · Graph REST| API
  API -.->|SMTP / Graph sendMail| Gmail

  API --> Postgres[(Postgres)]
  API --> MinIO[(MinIO<br/>attachments)]
  API --> Queue[(pg-boss<br/>jobs in Postgres)]
  API --> GitHub[(GitHub<br/>OAuth + Issues + Webhook)]

  Gmail -.->|delivers reply| Customer
```
